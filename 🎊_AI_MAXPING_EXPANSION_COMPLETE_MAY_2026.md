### 2. Using Curriculum Learning

```python
from ai.curriculum_learning import CurriculumManager

# Define levels
levels = [
    {'difficulty': 'easy', 'num_obstacles': 2},
    {'difficulty': 'medium', 'num_obstacles': 5},
    {'difficulty': 'hard', 'num_obstacles': 10}
]

# Create manager
curriculum = CurriculumManager(
    levels,
    success_threshold=0.7,
    window_size=100
)

# Training loop
for episode in range(1000):
    level_params = curriculum.get_current_level()
    
    # Train with current level
    success, reward = train_episode(level_params)
    
    # Update curriculum
    curriculum.record_episode(success, reward)
    curriculum.update()
```
