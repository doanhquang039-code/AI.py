# AI Dashboard Frontend

Angular 17 frontend for AI Training Dashboard.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```

## Features

- 🎯 Training Management
- 📊 Model Management
- 📈 Statistics & Visualization
- 🔄 Real-time Updates
- 🎨 Material Design UI

## API Integration

Backend API: `http://localhost:8000`

## Components

- Dashboard
- Training
- Models
- Statistics

## Tech Stack

- Angular 17
- TypeScript
- RxJS
- Chart.js
- Angular Material
